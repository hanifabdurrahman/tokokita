## 📖 Tentang Project

TokoKita merupakan aplikasi marketplace multi-toko berbasis web yang dikembangkan menggunakan **CodeIgniter 3** dan **MySQL**. Sistem ini memungkinkan setiap pengguna memiliki toko sendiri untuk mengelola produk, menerima pesanan, serta melakukan transaksi secara online.

Selain itu, aplikasi dilengkapi dengan integrasi pembayaran **Midtrans**, perhitungan ongkos kirim menggunakan **RajaOngkir Komerce**, serta panel administrasi untuk mengelola seluruh data marketplace.

## 🚀 Instalasi

Clone repository

```bash
git clone https://github.com/USERNAME/tokokita.git
```

Masuk ke folder project

```bash
cd tokokita
```

Import database

```
database/tokokita.sql
```

Konfigurasi database

```
application/config/database.php
```

Jalankan menggunakan Laragon / XAMPP.
